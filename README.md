# Project Name: ToDoList-Brief
================

## Overview
------------

A simple, interactive To-Do List application that helps users organize tasks efficiently by adding, editing, and deleting tasks. This app is built with HTML, CSS, and JavaScript for a streamlined user experience and easy task management.

## Features

- **Add Task**: Quickly add tasks with a single click or enter key press.
- **Edit Task**: Update tasks to keep information accurate.
- **Delete Task**: Remove tasks when completed or no longer needed.
- **Mark as Complete**: Check off tasks when done.
- **Responsive Design**: Usable on both desktop and mobile devices.

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling and layout
- **JavaScript** for functionality and interactivity

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/your-username/todolist.git
    ```
2. Open the project folder:
    ```bash
    cd todolist
    ```
3. Open `index.html` in your web browser to start using the app.

## Usage

1. **Add a Task**: Enter a task description in the input field and press the "Add" button.
2. **Edit a Task**: Click on a task to edit its content.
3. **Delete a Task**: Click the delete icon next to a task to remove it from the list.
4. **Complete a Task**: Click the checkbox to mark tasks as completed.
